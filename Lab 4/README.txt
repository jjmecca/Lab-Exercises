Functions:
1. definition: This function only takes one argument, the string of the word you want the definition of.
This function simply returns the definition of the word given.
2. pronounciation: This function only takes one argument, the string of the word you want the pronounciation of.
This function simply returns the pronounciation of the word given.
3. partOfSpeech: This function only takes one argument, the string of the word you want the partOfSpeech of.
This function simply returns the part of speech of the word given (noun, verb, adjective, etc.)
