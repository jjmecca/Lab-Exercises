How to use the Dictionary Wrapper:
1. Create a new file inside the same folder dictionaryapi.py is in.
2. Use "from dictionaryapi import DictionaryWrapper" to import it
3. Use functions like this: "DictionaryWrapper().[FUNCTIONNAME](word)"

Functions:
1. definition: This function only takes one argument, the string of the word you want the definition of.
This function simply returns the definition of the word given.
2. pronounciation: This function only takes one argument, the string of the word you want the pronounciation of.
This function simply returns the pronounciation of the word given.
3. exampleUsage: This function only takes one argument, the string of the word you want the example of.
This function simply returns an example of the word given being used.
4. partOfSpeech: This function only takes one argument, the string of the word you want the partOfSpeech of.
This function simply returns the part of speech of the word given (noun, verb, adjective, etc.)
