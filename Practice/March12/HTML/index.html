In this lab series you will learn the basics of HTML. What they are, what they do, and what the code looks like.

A basic understanding of these languages will help you know where to start when customizing the look of a web tool you're employing.

By the end of this unit I hope you'll be able to:

  1. Understand HTML
  2. Be abke to build out a basic shell of a website
  3. Recognize the look and feel of HTML
  4. Be able to webscrape
